# 🧮 Calculator + ⏱ Timer Web App

A simple and stylish **Calculator and Timer Counter** built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic DOM manipulation, event handling, and real-time updates in JavaScript.

---

## 🚀 Features

### 🧮 Calculator
- Basic arithmetic operations (+, -, *, /)
- Clear (C) button
- Delete (←) last input
- Decimal support
- Instant result using `=` button

### ⏱ Timer Counter
- Start, Stop, and Reset functionality
- Real-time HH:MM:SS display
- Continues counting every second when started

---

## 🛠 Technologies Used
- HTML5
- CSS3 (Glassmorphism UI design)
- JavaScript (Vanilla JS)
